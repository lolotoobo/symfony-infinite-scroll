## Ce que fait ce projet

Un flux de posts en défilement infini, sans rechargement de page, sans framework JS lourd.
Toute la logique métier reste en PHP. Le JavaScript se résume à un controller Stimulus.

---

## Lancer le projet

```bash
# Dépendances PHP
composer install

# Démarrer le serveur
symfony server:start
```

Ouvrir [https://localhost:8000](https://localhost:8000).
